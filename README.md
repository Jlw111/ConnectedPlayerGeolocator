# Connected Player Geolocator
Minecraft server plugin to print player geolocation upon joining

For the plugin to work, you will need to obtain an API Token from IPInfo.

This can be done at [https://ipinfo.io/signup](https://ipinfo.io/signup).

Paste your token into the `plugins/ConnectedPlayerGeolocator/config.yml` file that gets generated on first run, and then restart your server.
